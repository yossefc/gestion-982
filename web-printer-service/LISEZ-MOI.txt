INSTALLATION RAPIDE (5 minutes):
═════════════════════════════════════════════════════════════

1. INSTALLER NODE.JS
   ─────────────────
   → Allez sur: https://nodejs.org/
   → Téléchargez la version LTS (bouton vert)
   → Installez (cliquez "Suivant" partout)
   → REDEMARREZ l'ordinateur

2. VERIFIER L'IMPRIMANTE
   ──────────────────────
   → Connectez votre imprimante
   → Définissez-la comme imprimante par défaut:
     Windows → Paramètres → Imprimantes → Clic droit → Par défaut

3. COPIER LES FICHIERS
   ───────────────────
   → Copiez ce dossier vers: C:\Impression-Gestion-982\

4. INSTALLER LES DEPENDANCES
   ──────────────────────────
   → Ouvrez CMD dans le dossier (Shift + Clic droit → "Ouvrir PowerShell ici")
   → Tapez: npm install
   → Appuyez sur Entrée
   → Attendez 2-3 minutes

5. LANCER LE SERVICE
   ─────────────────
   → Double-cliquez sur: DEMARRER-SERVICE.bat
   → Gardez la fenêtre OUVERTE
   → C'est prêt! Les PDFs s'impriment automatiquement!
