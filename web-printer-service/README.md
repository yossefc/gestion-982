## ⚙️ הגדרות דפדפן (חשוב!)

### Google Chrome (מומלץ)

1. פתח את Chrome
2. עבור אל `chrome://settings/content/popups`
3. הוסף את `file://` לרשימת אתרים מורשים
4. פתח את `chrome://settings/content/notifications`
5. אפשר התראות

### Firefox

1. פתח את Firefox
2. הגדרות → פרטיות ואבטחה
3. אפשר חלונות קופצים מ-`file://`
4. אפשר התראות
